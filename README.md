# 🧠 双子星 - OpenClaw + Hermes 协同平台

双系统协同工作平台，OpenClaw 负责规划和决策（大脑），Hermes 负责执行（手脚）。

## 🎯 核心功能

- **左右分屏**: 左侧 OpenClaw（规划/审查），右侧 Hermes（执行）
- **标签页切换**: 规划/执行/审查/记忆/日志
- **共享记忆**: 向量数据库打通两个系统的知识和记忆
- **任务流**: 规划 → 执行 → 审查 完整流程
- **左侧边栏**: 任务列表管理

## 🚀 快速开始

### 前置要求

- Node.js >= 18
- npm >= 9

### 安装

```bash
# 克隆仓库
git clone git@github.com:guoyu767344855/shuangzixing.git
cd shuangzixing

# 安装前端依赖
cd frontend
npm install

# 安装后端依赖
cd ../backend
npm install
```

### 启动

```bash
# 启动后端服务
cd backend
npm run dev

# 启动前端 (新终端)
cd frontend
npm run dev
```

访问 http://localhost:5173 使用系统。
